# Decisions

## ADR-001
- 背景：需要将 vibe-coding 规范“可安装化”，让任意项目能快速具备同一套流程与模板。
- 方案选型：以 Codex/Claude skill 形式提供安装指令，并内置 kit 资源与安装脚本完成复制。
- 取舍：
  - 选择“文件拷贝”而非“远程拉取”：降低外部依赖，提升可离线使用性。
  - 选择“默认不覆盖 + 可备份覆盖”：避免破坏目标项目已有约束与文档。
- 影响范围：新增 `skills/vibe-coding-skills`，并将必要模板与规范打包进 assets。
- 决策人：admin
